# Roblox-Darkblue
A free theme for Roblox, in css.


### Requirements:
Some way to inject CSS code, I.E [Stylus](https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne)
Roblox set to dark mode by default.
